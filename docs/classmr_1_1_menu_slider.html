Detailed Description

template<typename T>
class mr::MenuSlider< T >

Template Menu Slider item class.

Handles value incrementing and decrementing for numeric types (int, float, double, etc.)

Template Parameters
TNumeric type to be controlled (e.g. int, float, double)
